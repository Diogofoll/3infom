Versionamento

o git é um software que permite realizar versões de um projeto. O github desktop é um software que faz a ligação entre VS Code, Git e Github.

o github desktop permite fazer a sincronia entre o repositório local e o repositório remoto.

commit

é o processo de confirmação de uma versão, ou seja, a cada alteração é possivel confirmar a alteração.

Push

esse comando envia(sincroniza) as alterações realizadas no repositório local com o repositório remoto no Github.
